In order to make customed settings, make sure the turn off easy-install.

To prevent easy-install, from new VM wizard, choose  install operating system later rather than accept the default installation disc file, be sure that "Power on this virtual machine after creation" checkbox is not selected.

https://www.wikihow.com/Prevent-Easy-Install-of-Virtual-Machine-in-VMware-Workstation

It's recommended to take snapshot of the VM when the install is finised.
